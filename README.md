## 開発環境
Ruby 2.4.3  
Rails 5.1.2  
Bootstrap 4.0.0  

RubyGemsをインストール
```
$ bundle install
```
データベースへのマイグレーションを実行
```
$ rails db:migrate
```
テストを実行
```
$ rails test
```
サーバーを起動
```
rails server
```
